# Warframe Relic

Unofficial fan-made website for browsing currently available Warframe relics, Prime items, and Varzia Prime Resurgence content.

### Current Relics
- Displays available relics parsed from Warframe Wiki
- Newly added relics are marked with `NEW`
- Each relic includes links to source information

### Current Prime Parts
- Displays Prime Warframes and weapons obtained from current relics
- Items can be marked with `NEW` or `UPDATE`
- Detailed pages show Prime parts and their associated relics
- Supports:
  - Viewing available getting sources
  

### Varzia Rotation
- Displays current Prime Resurgence (Varzia) relic pool
- Based on the 28-day rotation cycle
- Automatic update at the end of rotation

### Home Page
Provides preview sections from all categories:
- 8 relics
- 8 Prime items
- 8 Varzia items
